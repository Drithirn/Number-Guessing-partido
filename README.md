# 🎯 Guess My Number Game

An interactive number guessing game built using vanilla JavaScript, HTML, and CSS.

The player has to guess a randomly generated number between **1 and 20** while trying to maintain the highest possible score. The game provides real-time feedback, dynamic UI updates, and a confetti celebration effect on winning 🎉

# 🎮 How to Play

1. Enter a number between **1 and 20**
2. Click the **Check!** button
3. The game will tell you:
   - 📈 Too High
   - 📉 Too Low
4. Guess the correct number to win 🎉
5. Try beating your highest score 🏆
